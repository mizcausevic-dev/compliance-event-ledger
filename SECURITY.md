# Security Policy

## Scope

This repository is a portfolio implementation and uses in-memory sample data only. No production secrets, customer records, or regulated data should be committed.

## Reporting

If you notice a security issue in the project structure or dependencies, open a private report through GitHub Security Advisories or contact through [Kinetic Gain](https://kineticgain.com/).

## Safe Usage Notes

- keep secrets out of source control
- use environment-specific configuration outside the repo if persistence is ever added
- review dependency alerts before merging update branches
